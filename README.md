# GreenTodo

A minimal aesthetic green to-do list widget for macOS Übersicht.

## Screenshot

![Screenshot](screenshot.png)

## Installation

1. Install Übersicht:
   https://tracesof.net/uebersicht/

2. Download this repo as a ZIP.

3. Unzip and move the `GreenTodo.widget` folder to:

   ~/Library/Application Support/Übersicht/widgets/

4. Click the Übersicht menu bar icon → Refresh Widgets.

## Features

- Add tasks
- Mark tasks complete
- Delete tasks from both task list and summary list
- Live progress bar
